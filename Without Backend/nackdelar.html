Nackdelar att Beakta

Initial kostnad

Uppstartskostnaden kan vara betydande på grund av utrustning, system och installationer som krävs.

Tekniskt kunnande

Kräver god kunskap om näringslösningar, pH-värden, och systemunderhåll. Felaktiga inställningar kan snabbt skada växterna.

Beroende av elektricitet

De flesta system kräver kontinuerlig strömförsörjning för pumpar och styrenheter. Strömavbrott kan vara förödande.

Känslighet för fel

Problem med näringslösning eller utrustning kan snabbt påverka hela odlingen om de inte åtgärdas omedelbart.

Begränsad buffertkapacitet

Mindre förlåtande än jordodling vid tekniska fel eftersom det finns mindre buffert i systemet.

Risk för sjukdomsspridning

I cirkulerande system kan sjukdomar och skadliga organismer snabbt spridas till alla plantor.

Löpande kostnader

Kontinuerliga utgifter för elektricitet, näringslösningar och filterbyte kan bli betydande.

Komplexitet

Större system kan vara komplexa att underhålla och felsöka, särskilt för nybörjare.
